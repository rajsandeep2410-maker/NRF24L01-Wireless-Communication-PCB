# 🚀 NRF24L01 Wireless Communication PCB

## 📌 Overview
This project demonstrates a wireless communication system using NRF24L01 modules and Arduino, designed with a custom PCB in KiCad.

The design includes both:
- Transmitter (TX) section
- Receiver (RX) section

placed on a single board for demonstration and learning purposes.

---

## 🔧 Features
- NRF24L01-based wireless communication  
- Arduino interface for control and processing  
- Custom PCB design using KiCad  
- Power regulation and decoupling components  
- Clearly separated TX and RX sections  

---


## ⚙️ Working Principle
- The transmitter sends data wirelessly using the NRF24L01 module  
- The receiver captures the signal and processes it using Arduino  
- The received signal can be used to control outputs like LEDs or motors  

---

## ⚠️ Important Note
The transmitter and receiver are implemented as independent systems on the same PCB for demonstration purposes.

- Blue airwires (ratsnest) represent logical connections from the schematic  
- These are intentionally left unrouted to maintain separation between TX and RX sections
